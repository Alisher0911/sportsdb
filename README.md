# sportsdb

The Flutter project, which shows sporting events.

## Getting Started

Tools used:

- State Management: BLoC + freezed
- Network: dio
- UI: google_fonts, flutter_svg, webview_flutter
- API: [TheSportsDB](https://www.thesportsdb.com/)
